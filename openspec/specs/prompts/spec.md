## Requirements
### Requirement: understand_spec prompt exists
The system SHALL provide an understand_spec prompt.

#### Scenario: Prompt returns messages
- **WHEN** understand_spec is called
- **THEN** it returns a messages array

### Requirement: compare_specs prompt exists
The system SHALL provide a compare_specs prompt.

#### Scenario: Prompt accepts two specs
- **WHEN** compare_specs is called
- **THEN** it accepts spec_a and spec_b parameters
